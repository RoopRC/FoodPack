# **🍽️ FoodPack – Online Food Ordering Website**

---

## **📌 Project Overview**

FoodPack is a modern and responsive food ordering web application built using **HTML, CSS, and Vanilla JavaScript**.  
The website allows users to browse food items, explore categories, search for dishes, add items to a cart, and manage their order easily.

The project focuses on **interactive UI design, DOM manipulation, and frontend development concepts** without using any frameworks.

This application simulates a **real-world food delivery platform interface** similar to popular services.

## **✨ Features**

### **🍔 Food Menu**

Users can browse multiple food items with details like:

- Food name  
- Price  
- Restaurant  
- Delivery time  
- Ratings  

---

### **🔍 Search Functionality**

Users can search food items by:

- Name  
- Restaurant  
- Description  

Search results update instantly.

---

### **🍱 Category Filters**

Food items can be filtered using category buttons such as:

- All  
- Pizza  
- Burgers  
- Biryani  
- Desserts  
- Drinks  

---

### **🛒 Cart System**

Users can:

- Add items to cart  
- Increase or decrease quantity  
- View cart badge count  

Cart data is stored using **localStorage**.

---

### **❤️ Wishlist / Favorites**

Users can mark food items as favorites.

Features include:

- Toggle favorite icon  
- Save wishlist in localStorage  

---

### **🌙 Dark Mode**

The website includes a **dark mode feature**.

- Toggle between light and dark theme  
- Preference saved in localStorage  

---

### **🔔 Toast Notifications**

The application displays notifications for actions like:

- Adding item to cart  
- Removing from wishlist  
- Cart updates  

---

### **📱 Responsive Design**

The website works smoothly on:

- Desktop  
- Tablet  
- Mobile devices

## **🧠 JavaScript Modules**

### **App Controller (app.js)**

Handles the main application functionality:

- Rendering food items  
- Category filtering  
- Search functionality  
- Modal display  
- UI updates  

---

### **Cart System (cart.js)**

Manages cart operations such as:

- Adding items  
- Removing items  
- Updating quantities  
- Saving cart data in localStorage

### **Data File (data.js)**
